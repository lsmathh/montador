# MONTADOR
Program compiled on Linux using the command: 
```
g++ -o montador montador.cpp

```
## To run MONTADOR: 

```
./montador <file.asm>

```
# SIMULADOR
Program compiled on Linux using the command:
```
g++ -o simulador simulador.cpp

```

## To run SIMULADOR:

```
./simulador <file.obj>

```
* The simulator will only execute the next line after the user presses 'Enter'.
